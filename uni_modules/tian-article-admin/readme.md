# tian-article-admin
### 导入步骤
1. 合并页面配置，把**pages.json.data**内的subPackages内容，复制到pages.json里的subPackages下，与原有的配置平级
2. 右键上传database内opendb-news开头的schema
3. 运行项目，从菜单管理内的“**待添加菜单**”，导入文章所需要的菜单
